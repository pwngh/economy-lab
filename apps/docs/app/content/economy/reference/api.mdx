The library ships a curated, faceted [`exports`](https://nodejs.org/api/packages.html#exports) map: the main package re-exports everything, and each subpath is a focused entry point. Install it, then import from whichever one you need.

```bash
npm install @pwngh/economy-lab
```

```ts
import { createEconomy, topUp, toAmount } from '@pwngh/economy-lab';
import type { Store } from '@pwngh/economy-lab/ports';
```

## Focused entry points
